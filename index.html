Hear test

How well can you hear?

  • Use headphones and crank up the volume
  • Mute all distractive applications
  • Hit the start button and start listening
  • Hit any keyboard button when you hear a tone
  • Test results will appear once the test is completed

About this test

The test is specialized in measuring relative hearing sensitivity in the frequency domain. It will help to identify possible hearing loss in particular frequency range. Unless carefully calibrated the absolute sensitivity levels will not be reliable.

Some of the major factors affecting the test's accuracy:

  • Frequency response of headphones and the audio card
  • There's no easy way to calibrate the absolute intensity level of the test tones. This leads to systematic error in the end results.
  • Possible ambient noise
  • All frequencies are tested with equal intensity while ears have non-linear equal-loudness contour

This test is powered by the Web Audio API.
